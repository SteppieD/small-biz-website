## Image Management

The easiest customization is changing website images. All images are defined in simple HTML templates.

**See:** [IMAGE-MANAGEMENT.md](IMAGE-MANAGEMENT.md) for complete instructions on:
- Changing the owner/logo photo
- Updating gallery before/after images
- Finding Media Library URLs
- Troubleshooting image issues

**Quick Start:**
1. Upload images to WordPress Media Library
2. Copy the image URL
3. Edit `wp-content/themes/small-business-pro/index.php`
4. Replace the image URLs (they're clearly marked with comments)
5. Upload the updated file via FTP
