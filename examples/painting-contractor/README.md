# Painting Contractor Example

This is a complete, ready-to-use example of the Small Business Pro template configured for a painting contractor business.

---

## What's Included

- **Business Configuration** - Business details, contact info, service areas
- **SEO Keywords** - Keyword research and strategy for painting contractors
- **Image Customization Guide** - Step-by-step instructions for changing images

---

## Quick Start

### 1. Copy the Configuration Files

Copy these files to your WordPress installation:

```
examples/painting-contractor/business-config.php
  → your-site/config/business-config.php

examples/painting-contractor/seo-keywords.json
  → your-site/config/seo-keywords.json
```

### 2. Customize Business Information

Edit `config/business-config.php` and update:
- Business name
- Owner name
- Phone number
- Email address
- Service areas
- Business address

### 3. Change Images

Follow the complete guide: [IMAGE-CUSTOMIZATION.md](IMAGE-CUSTOMIZATION.md)

Quick overview:
1. Upload 7 photos to WordPress Media Library
2. Edit `wp-content/themes/small-business-pro/index.php`
3. Replace image URLs (clearly marked with comments)
4. Upload updated file

### 4. Review SEO Keywords

Check `config/seo-keywords.json` for recommended keywords for:
- Local area (city/region)
- Service types (interior, exterior, cabinet painting)
- Target keywords for search engine optimization

## Service Areas

This example is configured for a painting contractor serving:

- West Kelowna
- Okanagan region

Update the service areas in `business-config.php` to match your actual service territory.

## Image Customization

The website needs 7 images:

1. **Owner Photo** - Contractor headshot
2. **Interior Before** - Room before painting
3. **Interior After** - Same room after painting
4. **Cabinet Before** - Cabinets before refinishing
5. **Cabinet After** - Cabinets after refinishing
6. **Exterior Before** - House exterior before painting
7. **Exterior After** - House exterior after painting

See [IMAGE-CUSTOMIZATION.md](IMAGE-CUSTOMIZATION.md) for detailed instructions on:
- Where to find each image in the code
- Image requirements and dimensions
- How to get Media Library URLs
- Before/after composition tips
- Quality guidelines
